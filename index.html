    • Rename for the Web

    Remove spaces from a file name in Windows Explorer

    Project maintained by ancillaryfactory Hosted on GitHub Pages — Theme by mattgraham

    What it does

    Inspired by the Save for Web naming convention in Photoshop, right-click on the name of a file in Windows Explorer and select "Rename for the Web". Spaces in the filename will be replaced with hyphens.

    Installation

    1. Unpack the .zip file

    2. Run setup.bat

    3. Or, drag a file from Windows Explorer onto the Webname.exe icon

    NOTE: The installer has been tested in Vista and Windows 7, but it sometimes fails in XP. If the installation fails for any reason, you can still drag any file onto the Webname.exe icon to clean the filename. Deal.
