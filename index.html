    • Filename Slugifier

    Remove funny business from a file name in Windows Explorer

    my stupid, extra long filename?.pdf -> my-stupid-extra-long-filename.pdf

    What it does

    Inspired by the Save for Web naming convention in Photoshop, right-click on the name of a file in Windows Explorer and select "Convert to slug". Spaces in the filename will be replaced with hyphens, and the new filename will be copied to the clipboard.

    Installation

    1. Download and run install.exe

    2. Or, drag a file from Windows Explorer onto the Webname.exe icon

    3. That's all it does. Fer reals.
